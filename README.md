Dashboard de filmes (Movies dashboard) (Python, Streamlit e Pandas)

Dashboard de filmes feito para análise de dados com dataset importado do IMDb

Stack:

- Python
- Pandas
- Streamlit

Como rodar o projeto:

- python -m venv venv (ambiente virtual)
- pip install -r requirements.txt
- streamlit run app.py
